# 排序算法比较小工具（Win32 C++）

## 项目简介

这是一个基于 Windows 原生 Win32 API 的小工具，用来比较多种经典排序算法的性能。  
可以输入数据规模、随机种子和取值范围，程序会生成同一组随机数据，分别用不同排序算法进行排序，统计比较次数和移动次数，并以表格和折线图的形式展示结果。

项目比较简单，主要用于数据结构与算法课程的练习和演示。

---

## 技术栈

- 语言：C++（C++14）
- 平台：Windows
- 框架 / API：
  - Win32 API（对话框、按钮、编辑框、消息循环）
  - GDI（在新窗口中绘制折线图）
- 开发工具：
  - Visual Studio（生成 `.vcxproj` 工程）
  - Git / GitHub（代码托管）

---

## 主要功能

### 1. 参数输入

主界面提供三个输入框：

- `N(元素个数, 1-100)`：数据规模，最大 100
- `Seed(随机种子)`：用于控制随机数生成，可复现实验
- `Range(取值范围)`：随机数取值范围 `[0, range)`

默认值：

- N = 50
- Seed = 12345
- Range = 1000

### 2. 支持的排序算法

使用同一组随机数据，分别调用以下排序算法进行排序：

- 冒泡排序（Bubble Sort）`B`
- 插入排序（Insertion Sort）`I`
- 选择排序（Selection Sort）`S`
- 快速排序（Quick Sort）`Q`
- 希尔排序（Shell Sort）`H`
- 堆排序（Heap Sort）`HP`

每种算法都会统计：

- 比较次数：`comps`
- 移动次数：`moves`

通过一个 `Stats` 结构体记录，并在排序函数内部自动累加。

### 3. 实测结果输出（表格形式）

右侧文本框使用等宽字体（Consolas），以表格形式输出：

- 实验参数：`n / seed / range`
- 算法缩写说明：`B、I、S、Q、H、HP`
- 实测表格（当前随机数据）：


### 4. 折线图展示

提供一个“折线图”按钮：

- 需要先点击“运行”生成一组统计数据。
- 点击“折线图”后，会弹出一个新的窗口。
- 新窗口中绘制两条折线：
  - 蓝色：比较次数（comps）
  - 红色：移动次数（moves）
- X 轴上 6 个点分别表示：`B, I, S, Q, H, HP`

折线图方便直观对比不同排序算法在当前参数下的相对性能。

## 使用方式（简要）

1. 在 Visual Studio 中打开解决方案 / 工程。
2. 编译并运行程序。
3. 在主界面中：
   - 设置 `N / Seed / Range`
   - 点击“运行(&R)”查看右侧表格中的统计结果和理论复杂度说明
   - 点击“折线图(&L)”查看折线图展示
   - 点击“清空输出(&C)”清空右侧输出框
   - 点击“关闭”退出程序

---

## 适用场景

- 作为数据结构与算法课程中排序算法章节的实验小工具。
- 对比不同排序算法在同一组数据上的比较次数和移动次数。
- 辅助理解理论复杂度与实际运行统计之间的关系。

项目本身不复杂，更多是作为学习 Win32 编程和排序算法分析的一个小练习。  
如果你有兴趣，也可以在此基础上继续扩展更多排序算法或可视化方式。
